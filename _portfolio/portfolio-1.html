MedBook

A Health and Fitness app

The primary feature of the app is to store and manage Medical records and documents in a structured format over a cloud server i.e. Microsoft Azure. The records can be accessed from anywhere using the user's credentials.

More information at the GitHub repo

Low Cost Braille Printer

The purpose of this project was to introduce a portable and low cost Braille printer. This project was funded by MAHE. We developed two prototypes - the first with an emphasis of cost-effectiveness, and the second prototype, which is more experimental and aims to eliminate several hurdles of Braille printing.

Technical Publication: IEEE Xplore

Media mention: Times Of India
